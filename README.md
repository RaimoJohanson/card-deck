# Prerequisites
- Docker
- Node.js
# Setup
- Install dependecies `npm install`
- Start database `sh src/scripts/start-db.sh`
- .env file is added in source control for easy demo purpose

# Run

- Run app `npm run start`
# Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
